# BJeans.co

"Your Denim, Your Way"

Initial workspace configuration only (no feature implementation yet).

## Structure

- backend: Node.js + Express.js + MongoDB config skeleton
- frontend: Next.js + React + Tailwind config skeleton

## Quick Start

1. Install root dependency
   - npm install
2. Install dependencies for both apps
   - npm run install:all
3. Run both apps in development mode
   - npm run dev

## Notes

- This repository currently contains configuration and folder scaffolding only.
- Business modules and API implementation are intentionally not created yet.
