# Authentication & RBAC Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement a secure authentication system with role-based access control (RBAC) using MongoDB and JWT in HttpOnly cookies.

**Architecture:** We will use Next.js App Router API Routes for auth, Mongoose for user models, and middleware for session/role validation.

**Tech Stack:** Next.js 16, TypeScript, MongoDB (Mongoose), JSON Web Tokens (JWT).
